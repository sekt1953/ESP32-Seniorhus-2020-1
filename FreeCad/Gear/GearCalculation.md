# Gear Calculation:

CP 	~ Pitch  
PCD ~ Pitch Circle dia.  
DP	~ Diameter Picth  
m	~  
Addendum ~ Outside Circle dia.  
Deendum	~ Inder Circle   
C1	~  
C2	~  
N 	~ Number of Tooth  

PCD 	= m * N   
DP	= N / PCD  
CP	= (PCD * π) / N  ~  CP =  π / DP  
m	= PCD / N  
  
Addendum	= CP / π  
Clearance	= CP / 20  
Add. Circle dia.	= PCD + ( 2 * Addendum )  
Dedendum	= Addendum + Clearence	  
Dedendum Circle dia.	= PCD – ( 2 * Dedendum )  
Tooth thicknes	= CP / 2
